# C++/SDL2 Mandelbrot
+ CPU bound
+ a couple optimizations: border tracing, multithreading (**OpenMP**) and SIMD (**AVX2**)
  _(if supported by your hardware)_
+ **Zoom in/out with mouse wheel**
+ **Increase/decrease max iterations by pressing UP/DOWN**
